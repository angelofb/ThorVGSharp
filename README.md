# ThorVGSharp

Low-level .NET bindings and high-level wrapper classes for the [ThorVG](https://github.com/thorvg/thorvg) vector graphics library.

## Features

- P/Invoke bindings for the ThorVG C API
- High-level managed wrapper classes for canvas, shapes, and text rendering
- Cross-platform native libraries for Linux, macOS, Windows

## Installation

```
dotnet add package ThorVGSharp
```
